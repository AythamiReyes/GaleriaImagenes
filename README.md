# GaleriaImagenes.

Galeria Interactiva Desarrollada con HTML, CSS3 y JavaScript puro.

Nos muestra unas cuadriculas con imagenes, al hacer click se nos abre una ventana y dentro se puede navegar entre imagenes. Añadimos las flechas para una vez clickada la imagen poder verlas en orden, de izquierda a derecha, o poder seleccionar la imagen que queremos ver.

Tecnologias Usadas.

HTML5, CSS3, JavaScript.
